## Description

This project implements and compares Genetic Algorithm (GA) and Iterated Local Search (ILS) for solving the 0/1 Knapsack Problem. The code runs both algorithms on a set of benchmark datasets, collects statistical results, and performs a Wilcoxon signed-rank test to compare their performance.

## How to Run

1. Ensure you have Java installed (Java 8 or higher).
2. Place all dataset files in the project directory.
3. Compile the code:
        - javac *.java
4. Run the program:
        -java Main
5. When prompted, enter a seed value (any integer).
6. Alternatively, run java -jar KnapsackSolver.jar

The program will output the results for each dataset and the statistical comparison.
